配置JDK环境

  1. 安装JDK，本教程选用JDK8，大家自行下载安装安装即可！
  2. win+R，输入cmd打开cmd命令界面
  3. 输入java -version，查看JDK版本
  4. 配置系统环境，按图即可
  5. …
  6. …
  7. 选择系统变量里的新建
  8. 变量名设置为:JAVA_HOME 变量值为JDK的安装目录,一般默认目录都一样，按着图片地址找就行了
  9. 选择系统变量里的Path，点击编辑或者双击都可
  10. 添加两个变量：%JAVA_HOME%\bin %JAVA_HOME%\jre\bin
  11. 在cmd界面，输入命令javac，如图输出即可

安装Eclipse

1.下载Eclipse

  • 百度进入Eclipse官网,首页默认下载的是最新的64位2020.09版本，我们本篇讲2020.03版的安装，因为2020.09版不支持直接使用JDK8，选择箭头指的地方
  • 进入后在右下方选择2020.03版本
  • 有两种方法下载，第一种下载选择包，第二种直接下载包，这里主要介绍第一种

  • 点击下载后进入此界面，注意更改下载地址为中国的地址

  • 两种方式的下载速度都是不错的
    第一种，选择包，选择后还需在线下载，因此较小,后面有详细步骤：

    第二种，完整包，所以比较大，解压后点击.exe即可：

2.安装Eclipse

  • 如上面步骤提示，因为我走web，所以这里我选择Eclipse IDE for Enterprise Java Developers 这个版本以前叫做 Eclipse IDE for Java EE Developers，面向企业级开发

  • 进入2020.03版本安装界面

    如果是2020.09版本则无法选择JDK8，应该只能预装JDK11及以上版本，但是如果本机并未预装则会因为下载过慢而导致安装失败！如图所示：

  • 等待安装完成即可，出现黄色提示框为下载速度较慢，耐心等待即可

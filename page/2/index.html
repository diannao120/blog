电脑故障及解决办法

发表于 2019-11-14
作者：18级 郭尊

1 电脑WiFi连不上

找到“网络”，选择“属性”
找到“更改无线网络”，左键进入
下面罗列的都是连接过的无线网络历史，右键想要删除的，选择“删除网络”就ok了

2 U盘识别不了

U盘读不出来，又有很多种情况，可以具体情况具体处理：
U盘插入电脑，未有任何反应，
在计算机管理——磁盘管理中，未发现U盘设备
这种情况，一般是U盘坏掉了，不上电与识别。
在计算机管理——磁盘管理中，发现U盘设备
这时需要为U盘添加个盘符（右键菜单-添加盘符），才可显示并使用
U盘插入电脑，有反应，
提示未识别设备
这种情况，一般是U盘坏掉了，晶振或主控故障，无法正确识别。
有盘符，提示无法打开或未格式化
这时，需要在在计算机管理——磁盘管理中。看看U盘的大小是否正常；
如果不正常，说明存储介质出问题了，需要重新量产才能修复；
如果大小正常，说明是分区错误，可用Winhex工具-打开磁盘，查看U盘上的分区及文件系统，需要修复分区错误后使用

电脑120值班报告

发表于 2019-11-14
作者：18级 金志礼

1、 装系统

打开启动盘，插入电脑，把自动启动关了，安全固定打开，进入大白菜界面，快速分区，win10选择长的，win7选职责短的，然后重装，根据情况选择给哪个打勾，点击重启后拔出U盘

2、 装office

把office压缩包复制到别人电脑里，然后解压到E, set up,撞到想装的地方就好了

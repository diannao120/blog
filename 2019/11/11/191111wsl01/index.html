常见电脑问题（2019.11月之前）

发表于 2019-11-11 更新于 2019-11-12
作者：18级 王少磊

问题1：

win10系统开机不是进入系统而是出现蓝屏界面，并提示错误代码0xc0000428。

解答1：

开机按shift+重启键进入高级启动界面，点击“疑难解答”，点击打开“高级选项”，在“高级选项”界面，点击“启动设置”，在“启动设置”界面里，找到并点击“重启”，然后点击“禁用驱动程序强制签名”即可。

问题2：

华硕笔记本BIOS风扇设置。

解答2：

开机按DEL进BIOS，在左面一列找PC health，进入该列表页，找smart fan（也就是温控风扇的意思），有两项MAX或者其他，MAX就是一直最高速度，另一项可以设置温度叫做target temperature叫目标温度，在达到目标温度前，风扇会满速运转，直到温度达到目标温度以下。

问题3：

win7系统开机出现grub，表示主引导文件丢失或损坏。

解答3：

一、修复主引导：
方法一：使用DOS命令修复
1、在grub命令提示符后面输入以下命令（不包含grub〉本身）：
grub> rootnoverify (hd0,0)
grub> chainloader +1
grub> boot
2、这样可以进入到windows系统里，然后按下win+R，再输入cmd，进入命令窗口，再输入：fdisk /mbr回车即可。
方法二：使用专用工具修复
1、使用启动U盘或光盘启动电脑，进入PE系统
2、运行DISKGENIUS工具，重建主引导记录
二、重装系统：若以上方法均无效，则需要重装系统

问题4：

win10系统重启死循环（可能是系统显卡出现问题）

解答4：

1、将电脑开机，在开机界面出现时按住F8，进入系统的设置 界面
2、找到“更新和安全选项卡”，点击进入后，打开“恢复选项卡”
3、然后点击“高级启动”中的“立即重启”按钮，进入修复界面
4、进入修复界面后，点击“高级选项”，点击“疑难解答”
5、点击“高级选项”
6、点击“启动选项设置卡”，进入后打开启动设置
7、在此页面中点击下方的重启按钮，把电脑进行重启操作
8、这时电脑会弹出“启动安全模式选项卡”，在此页面直接按F4（可能每台电脑进入的按键不同）进入电脑的安全模式
9、电脑重启完毕后会自动进入安全模式，然后点击开始菜单，使用鼠标右键点击菜单中的设备管理选项卡
10、在设备管理窗口中找到显卡驱动程序，然后鼠标右键点击，将显卡驱动程序禁用
11、禁用完成后重启电脑，这时电脑会正常开机进入主界面，然后重新下载安全显卡驱动程序即可

问题5：

win10系统开机输入密码后黑屏

解答5：

1、打开电脑，输入开机密码进入
2、显示黑屏后，按住“Ctrl+Alt+Del”打开任务管理器
3、在打开的任务管理器中，点击“文件”，“运行新任务”
4、再弹出的“新建任务”窗口中输入“explorer”命令，点击确定
5、这时候成功进入桌面，但为了彻底解决黑屏故障继续以下操作；按住“Win+R”，在打开的输入框中输入“regedit”，按回车
6、在弹出的注册表编辑器中依次打开：
HKEY_Local_Machine —- Software —- Microsoft —- Windows NT —- CurrentVersion —- Winlogon选项
7、在右侧列表中双击“shell”项，在打开的编辑字符串窗口中修改数据数值为“explorer.exe”点击确定，保存后推出注册表即可

问题6：

电脑连接打印机后无法找到

解答6：

1、按住“Win+R”，在打开的对话框中输入“compmgmt.msc”，点击确定
2、在打开的新窗口左侧列表中展开“设备管理器”，在右侧的“串行总线控制器”中找到相应的打印机设备，确保此设备没有被禁用
3、展开左侧的“服务”界面，在右侧找到“Print Spooler”服务项，确保此服务处于正常开启状态
4、更多情况下造成打印机无法识别或打印的原因是相关的打印驱动未正常安装，对此，我们可以通过相关驱动管理器工具（例如驱动精灵、驱动人生等）进行检测及安装驱动

校园网客户端常见问题解决方法汇总

发表于 2019-11-10 更新于 2019-11-27 分类于 维修专栏 Valine：
作者：17级 薛西龙 （转）

问题1：

登陆后，弹出提示：发现您正在使用以下代理软件：Windows(vista)系统自带共享代理。

解决方法：

具体操作为，依次打开：控制面板 → 网络和 Internet → 网络和共享中心 → 更改适配器设置
→ （右键单击）本地连接 → 属性 → 共享 → 取消勾选“允许其他网络用户….来连接”

问题2：

无法登陆，弹出提示：“802.1认证错误：8021x登陆超时失败！”

解决方法：

在桌面右下角，打开网络连接列表。如果有无线网已经连接上，请断开连接，然后重新登陆客户端。

如果没有无线网络连接，仍然出现此问题，请关闭无线网卡后重新登陆。

问题3：

无法选择网卡或无网卡选择

解决方法：

1、关闭 “本地连接” 的共享选项即可。
具体操作为，依次打开：控制面板 → 网络和 Internet → 网络和共享中心 → 更改适配器设置 → （右键单击）本地连接 → 属性 → 共享 → 取消勾选“允许其他网络用户….来连接”
2、若共享早已关闭还出现此问题，请关闭客户端，在任务管理器里面结束所有关于wifi软件的进程，然后在客户端文件夹里面找到”修复客户端驱动程序“进行双击运行，最后重新登录客户端。
注：有时重启电脑也可解决此问题

问题4：

点击登陆后，一直处于正在登陆状态，无法成功登陆。然后提示“连接认证服务器超时”

解决办法：

（1）先禁用“本地连接”，再重新启用“本地连接”。（禁用和启用网卡的方法为：控制面板 → 网络和Internet → 网络和共享中心 → 更改适配器设置 → 右键单击“本地连接” → 禁用/启用）

（2）或者右键我的电脑(win7 是计算机) - 管理 - 服务与应用程序 - 服务 - Internet Connection Sharing (ICS) 重启服务即可。

注：重启电脑也可解决此问题，建议每次退出客户端时先注销再退出，保证正常退出

问题5：

客户端页面无法自动显示，通知栏出现两个客户端图标（一个彩色一个黑白）

解决方法：

关闭 “本地连接” 的共享选项即可。
具体操作为，依次打开：控制面板 → 网络和 Internet → 网络和共享中心 → 更改适配器设置 → （右键单击）本地连接 → 属性 → 共享 → 取消勾选“允许其他网络用户….来连接”

或者重新安装客户端

问题6：

控制端口被占用

解决方法：

先试试去掉本地连接共享.试过无效，在任务管理器中结束掉ishare_user.exe；如果找不到此项程序，进行第三个方法，打开菜单栏的运行，输入“cmd”出现DOS窗
再 输入“netsh winsock reset”,点击Enter
系统会运行Windows目录修改完毕，把计算机重启后即可！

问题7：

选择网卡列表里有N多网卡供选择

解决方法：

1、正常情况下，选择网络适配器（网卡）对话框中会出现使用的网卡类型，如Realtek网卡。直接选择使用有线连接的网卡，点击选择网卡登陆即可。
2、如系统存在多个网卡或者安装了VMware Workstation等虚拟机软件，建议“禁用其他网卡”和“虚拟机生成的网卡”。
3、XP系统：打开“控制面板”-》“网络连接”，除“本地连接”外，把其余的连接能删除的就删除，不能删除的禁用.
4、WIN 7系统：打开“控制面板”-》“网络和共享中心”，点击左侧的“更改适配器设置”，除“本地连接”外，把其余的连接能删除的就删除，不能删除的禁用(需要用WiFi的话无线网卡和虚拟无线网卡不要删除和禁用)。

问题8：

客户端闪退

解决方法：

打开客户端文件夹 点击修复客户端驱动程序，重新开启客户端

问题9：

wifi代理软件导致断网

解决方法：

关闭所有wifi软件进程，或卸载wifi软件，适当时候需要重启电脑才能登录客户端

问题10：

内网认证没有发现Dr.com扩展数据

解决方法：

这个情况让我也棘手了好久，因为本人没有遇到这种情况，至今还没有确切的办法，但是可以提供几个方向：
1、可能是客户端文件丢失，试试删除已安装客户端，重启笔记本，安装客户端
2、可能xp用户系统配置不完全，没有插件支持该客户端（win7不存在此情况），xp用户最好把系统换成win7，不要换win8（兼容性较差）
3、可能某些杀毒软件报毒，删除了某些文件。
